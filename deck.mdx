# Base LEGI

LEGI est exhaustif pour les lois, décrets-lois, ordonnances, décrets depuis 1945. 73 codes officiels en vigueur consolidés +29 abrogés.

- 1.3 Go
- 110 tar.gz
- 2.5 Millions de fichiers XML
- MAJ Quotidienne vers 20h
- des anomalies : http://anomalies.legilibre.fr

API prévue au printemps 2019
